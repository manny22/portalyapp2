Mejora el frontend de Janora para que tenga apariencia de producto SaaS comercial.

Usa React, TypeScript y Tailwind CSS.

Quiero una interfaz moderna, limpia, responsive y profesional para administración de copropiedades.

Mejora:

- Layout principal.
- Sidebar.
- Header.
- Dashboard.
- Tablas.
- Formularios.
- Modales.
- Cards.
- Estados vacíos.
- Estados de carga.
- Estados de error.
- Página de control de acceso para vigilantes.

La pantalla de control de acceso debe ser extremadamente práctica para una portería:

- Búsqueda grande por documento, placa o código de autorización.
- Botones claros para registrar ingreso y salida.
- Alertas visuales de autorizado, expirado, denegado o ya ingresado.
- Listado de personas actualmente dentro.
- Diseño usable en tablet o computador de portería.

Mantén Tailwind CSS, componentes reutilizables y código limpio.
